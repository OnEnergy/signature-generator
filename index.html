How to use in Gmail
  1. Click Copy Signature.
  2. Open Gmail settings ➜ See all settings ➜ Signature.
  3. Create a new signature and paste the copied content.
  4. Choose the new signature for Default on new emails and replies/forwards.
  5. Save changes.
